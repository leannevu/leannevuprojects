About This Project

Why I built this

I built this dashboard to bring my study workflow into one place: focused work sessions, daily time logs, notes, and reference material. Instead of tracking progress in separate files and tabs, the dashboard turns those habits into a single interface I can return to every day.

The project also gave me a chance to practice building a more complete Flask application: CSV-backed data, dynamic calendar rendering, persistent notes, import tools, and a PDF viewer all working together in one user-facing dashboard.
