Portfolio Demo

Optimized Portfolio Tool

A lightweight portfolio optimization demo that generates stock allocations and performance metrics, then visualizes the resulting portfolio for a more interactive financial analysis experience.

Personas

CICurious Investor

Wants a quick allocation example with readable metrics.

DLData Learner

Connects optimization logic to visible portfolio output.

PRPortfolio Reviewer

Checks allocation, risk, and return tradeoffs.

User Stories

1 Select Universe

Start with available stock data.

2 Generate Portfolio

Run allocation logic through the demo.

3 Review Allocation

See which tickers receive portfolio weight.

4 Compare Metrics

Read return, volatility, and Sharpe ratio.

5 View Chart

Inspect the generated performance visual.

6 Randomize Again

End by comparing another portfolio scenario.
